# Enterprise Cybersecurity Awareness & Threat Mitigation Framework

## Project Overview
This repository contains the strategic framework for a comprehensive Cybersecurity Awareness Training Plan designed to mitigate digital threats, fortify enterprise security posture, and educate associates on user-level vulnerabilities. This project was developed as part of a professional virtual engagement with Paladin Consulting LLC via the Climb Hire / Riipen IT Support & AI Pathways program.

## Core Competencies Demonstrated
* **Security & Compliance:** Vulnerability Assessment, Threat Mitigation, Cybersecurity Awareness Training, IT Security Best Practices.
* **Core Competencies:** System Analysis, Technical Documentation, Phishing Vectors Research.

---

## Key Achievements & Deliverables

### 1. Cybersecurity Awareness Training Plan
* Developed targeted enterprise training modules focusing on human-layer defense architectures.
* Formulated actionable strategies for identifying and neutralizing phishing, social engineering, and unauthorized access vectors.

### 2. Threat Modeling & Vulnerability Assessment
* Conducted system-level threat modeling to map infrastructure risks.
* Aligned user training modules with modern defense-in-depth principles to reduce organizational attack surfaces.

## Technical Artifacts
* `phishing_simulation.py`: A Python automation script designed to simulate an internal phishing educational campaign, helping IT administrators log and analyze simulated employee click-through metrics for risk mapping.
